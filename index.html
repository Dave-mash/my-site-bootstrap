PROJECTS

Binary Rain

Binary Rain is an inspiration of the Matrix Rain by Emily Xie, a creative coder based in the USA. The work was done in P5, a Javascript front-end framework for building quick sketches/visuals. The main objective of the project was to visualize binary symbols trickling from top to bottom. Through this project, I found my love for front-end visualizations and I'm definitely going to continue exploring this art.

Flask Blog

Flask Blog is a simple blogging application originally built in Flask, vanilla Javascript and plain css. This work is inspired by Corey Schafer's(a professional coder and instructor) Blogging app in Flask youtube video. The main goal for this project was to cover all basic CRUD operations.
